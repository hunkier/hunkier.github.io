冒泡排序

发表于 2019-11-22 分类于 java ， algorithm ， bubble sort Disqus：

(1) 比较前后相邻的二个数据，如果前面数据大于后面的数据，就将这二个数据交换。

(2) 这样对数组的第 0 个数据到 N-1 个数据进行一次遍历后，最大的一个数据就 ”沉“ 到数组di N-1 个位置。

(3) N=N-1，如果 N 不为 0 就重复前面二步，否则排序完成。
